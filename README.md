# 질문있는학교 선도교원 연수 — 강사 일정표

충남교육청 「질문있는학교 선도교원 연수」 집합연수(1~10기) 강사 일정표 사이트입니다.
빌드 과정이 없는 순수 정적 사이트로, **이 저장소의 파일이 곧 소스**입니다. 파일을 고쳐서 push하면 그대로 배포됩니다.

## 구조

```
index.html                     페이지 뼈대 (게이트 + 앱 마크업)
robots.txt                     크롤러 수집 거부
assets/
  styles.css                   전체 스타일 (앱 + 접속 게이트)
  gate.js                      비밀번호 게이트 + 구글 시트 CSV 로드
  csv2data.js                  CSV → 일정 데이터 변환 (장소 좌표 포함)
  app.js                       일정표 렌더링 (과목별 표·검색·이동 지도)
  vendor/leaflet/              Leaflet 1.9.4 (지도 라이브러리, 수정하지 않음)
```

## 일정 데이터 수정

일정은 담당자의 **구글 시트와 실시간 연동**됩니다(주소는 `assets/gate.js`의 `CSV_URL`).
시트만 고치면 되고, 코드 배포는 필요 없습니다. 방문자가 새로고침하면 수 분 내 반영됩니다.

- 시트에서 게시(웹에 게시) 설정을 끄면 사이트가 데이터를 못 불러오니 유지해야 합니다.
- 연수 장소가 바뀌면 `assets/csv2data.js`의 `VENUES` 좌표·주소도 함께 수정합니다.

## 코드 수정과 배포

파일 수정 → commit → push 하면 GitHub Pages가 1~2분 내에 다시 배포합니다.
CDN 캐시 때문에 코드 변경 반영에 최대 10분쯤 걸릴 수 있습니다(일정 데이터는 캐시와 무관하게 항상 최신).

## 비밀번호 변경

`assets/gate.js`의 `PW_HASH`를 새 비밀번호의 SHA-256 값으로 바꿉니다.

```sh
printf '%s' '새비밀번호' | shasum -a 256
```
